# Programme Example — AMPEL360 BWB Q100 (BWB-Q100)

## Overview

The **AMPEL360 BWB Q100** is a regional clean-sheet Blended Wing Body (BWB) aircraft with hydrogen-electric hybrid propulsion for ~100 passengers. It represents the core hydrogen-aviation programme targeting regional routes with zero in-flight CO₂ emissions.
