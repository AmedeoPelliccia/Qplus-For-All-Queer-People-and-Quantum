# IDEALE-ESG Impact Study — AMPEL360 City eVTOL — UAM (City-eVTOL) — Overview

## 1. Programme Identity

| Field | Value |
|---|---|
| Programme | AMPEL360 City eVTOL — UAM |
| Abbreviation | City-eVTOL |
| Generation | Next-generation UAM |
| Aircraft type | electric vertical take-off and landing urban air mobility aircraft |
| Energy carrier | battery-electric / hybrid-electric |
| Study framework | IDEALE-ESG |

## 2. Study Purpose

This impact study evaluates a hybrid electric vertical take-off and landing aircraft designed for **Urban Air Mobility (UAM)** applications against the nine IDEALE-ESG axes:

| Axis | Name |
|---|---|
| A | Aerospace |
| D | Defense |
| E | Energy |
| E2 | Economics |
| E3 | Environments |
| G | Governance |
| I | Information |
| L | Logistics |
| S | Social |

## 3. Study Scope

The study:

- identifies impacts of the City-eVTOL programme across each IDEALE-ESG axis;
- scores each axis using the controlled impact scoring model (0–5);
- identifies cross-axis dependencies and compound risks;
- defines evidence requirements for lifecycle gate closure;
- provides programme recommendations per axis and in aggregate.

## 4. Governance Boundary

This study is a **programme-impact and governance artefact**. It is not a certified aircraft design, production approval, operational approval or legal compliance finding.
