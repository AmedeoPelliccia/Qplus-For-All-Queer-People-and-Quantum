## 3. Lifecycle Gate Structure

| Gate | Phase | Key IDEALE-ESG Axes | Closure Condition |
|---|---|---|---|
| Gate 0 — Programme Launch | Concept initiation | G, E2 | Programme charter approved |
| Gate 1 — Concept Freeze | Concept complete | A, E, G, E2 | Concept evidence package accepted |
| Gate 2 — Preliminary Design Review | PDR | A, E, E3, G, I | PDR evidence package accepted |
| Gate 3 — Critical Design Review | CDR | A, E, E3, G, I, L | CDR evidence package accepted |
| Gate 4 — First Flight | Flight test | A, E, G, S | Flight clearance granted |
| Gate 5 — Entry Into Service | EIS | All axes | Type certificate and EIS evidence accepted |
