# 05-03 — Propulsion and Thermal System Fault Prediction
## AMPEL360-PLUS Spacecraft Q10

---

## 1. Scope

This sub-section covers **neural network models for fault prediction in main propulsion, reaction control systems (RCS), and vehicle thermal management** for the AMPEL360-PLUS Q10.

It targets incipient fault detection, isolation, and prognosis for engines, propellant systems, RCS thrusters, and active thermal control loops.

---

## 2. Objectives

- Apply NN-based fault detection and isolation (FDI) to main engine and RCS thruster health monitoring.
- Predict propellant system anomalies (valve degradation, feed system contamination, seal wear).
- Monitor thermal management system (radiators, heat pipes, coolant loops) for degradation and anomaly prognosis.
- Generate maintenance alerts and replacement recommendations to ground operations.
