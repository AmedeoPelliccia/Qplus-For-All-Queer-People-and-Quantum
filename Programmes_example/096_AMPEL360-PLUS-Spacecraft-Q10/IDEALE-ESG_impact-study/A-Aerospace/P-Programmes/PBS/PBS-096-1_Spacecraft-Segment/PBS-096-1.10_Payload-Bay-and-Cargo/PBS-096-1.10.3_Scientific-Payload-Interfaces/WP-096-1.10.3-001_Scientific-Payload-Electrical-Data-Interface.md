# WP-096-1.10.3-001 — Scientific Payload Electrical and Data Interface

**Section:** PBS-096-1.10.3  
**Programme:** AMPEL360-PLUS Spacecraft Q10 (PBS-096)

## Purpose

Standard electrical, data, and thermal interface for hosted scientific payloads including power allocation, SpaceWire/MIL-1553 data, and vibration isolation.

## Scope

This work package defines the requirements, design approach, verification strategy, and evidence artefacts for scientific payload electrical and data interface.
