# Programme Example — AMPEL360 BWB H2 Demonstrator (H2-DEMO)

## Overview

The **AMPEL360 BWB H₂ Demonstrator (H2-DEMO)** is a scaled technology demonstrator for the BWB hydrogen-electric aircraft family. Its objective is to mature LH₂ propulsion, cryogenic storage, and BWB aerodynamic integration to TRL 5–6 ahead of the Q100/Q250 production programme.
