# D — Defense | I — Infrastructures | 03 — Test Ranges and Evaluation Facilities
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **test ranges and evaluation facilities** required for the AMPEL360 MRTT Q300 programme qualification, certification, and ongoing evaluation activities. It covers flight test centres, air-to-air refueling (AAR) trial ranges, systems integration labs (SILs), and operational evaluation (OPEVAL) infrastructure.

## 2. Scope

- Flight test centre requirements for MRTT-Q300 qualification
- Air-to-air refueling (AAR) trial airspace and receiver aircraft availability
- Boom, hose-and-drogue, and multi-point systems test infrastructure
- Systems integration laboratory (SIL) and iron bird facility requirements
- Operational evaluation (OPEVAL) and operational test and evaluation (OT&E) infrastructure

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| Flight test range | Is adequate restricted airspace and instrumentation available? |
| AAR trial infrastructure | Are receiver aircraft and boom/HDU test assets available? |
| SIL/iron bird | Is systems integration lab infrastructure identified? |
| Telemetry and chase | Is telemetry/chase aircraft infrastructure in place? |
| Certification authority access | Can airworthiness authority participate in trials? |
