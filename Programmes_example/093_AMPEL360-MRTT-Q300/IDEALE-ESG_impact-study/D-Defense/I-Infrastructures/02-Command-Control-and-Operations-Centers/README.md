# D — Defense | I — Infrastructures | 02 — Command, Control and Operations Centers
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **command, control, and operations center** (C2) infrastructure for the AMPEL360 MRTT Q300 programme. It covers the ground-based C2 nodes, mission planning facilities, and operational control centers that manage, coordinate, and direct MRTT-Q300 tanker and transport missions.

## 2. Scope

- Mission control and tanker task unit (TTU) operations center requirements
- Air refueling control team (ARCT) infrastructure and communications
- Airspace coordination and deconfliction infrastructure
- Mission planning system (MPS) integration at operations centers
- Secure voice and data communications architecture for C2 nodes

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| C2 node connectivity | Are operations centers connected to MRTT-Q300 comms architecture? |
| Mission planning infrastructure | Is MPS infrastructure adequate for multi-role mission planning? |
| Airspace management | Does C2 infrastructure support complex airspace coordination? |
| Redundancy and resilience | Are backup/alternate C2 nodes defined? |
| Interoperability | Can C2 nodes interface with coalition and civil ATC systems? |
