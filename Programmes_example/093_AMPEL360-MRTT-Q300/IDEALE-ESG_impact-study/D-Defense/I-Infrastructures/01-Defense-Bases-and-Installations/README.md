# D — Defense | I — Infrastructures | 01 — Defense Bases and Installations
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **defense bases and permanent installations** relevant to the AMPEL360 MRTT Q300 programme. It covers fixed main operating bases (MOBs), forward operating bases (FOBs), and contingency locations where the MRTT-Q300 will be based, operated, and supported.

## 2. Scope

- Main operating base (MOB) infrastructure suitability: runway length, pavement classification, ramp space
- Fuel and support infrastructure compatibility with Q300 class requirements
- Hangar, shelter, and maintenance bay availability
- Security perimeter and access control at basing locations
- Environmental and operational constraints at candidate bases

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| Runway / taxiway suitability | Does the base accommodate MRTT-Q300 MTOW and dimensions? |
| Fuel infrastructure | Are fuel systems compatible with SAF/hybrid-capable requirements? |
| Maintenance infrastructure | Are MRO facilities adequate for multi-role configuration? |
| Security posture | Does base security meet requirements for tanker/transport operations? |
| Basing agreements | Are host nation agreements in place or required? |
