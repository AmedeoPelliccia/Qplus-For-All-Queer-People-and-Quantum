# D — Defense | I — Infrastructures | 06 — Secure Communications and Data Infrastructure
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **secure communications and data infrastructure** for the AMPEL360 MRTT Q300 programme. It covers ground-based and space-based secure communications networks, classified data handling infrastructure, S1000D/CSDB data hosting, and quantum-enhanced communications architecture.

## 2. Scope

- Secure voice and data communications networks (HF/VHF/UHF/SATCOM/MILSATCOM)
- Encrypted communications infrastructure (NATO/national crypto standards)
- Mission data loading and ground-based mission data system (MDS) infrastructure
- S1000D/CSDB technical data hosting on secure networks
- Quantum-enhanced communications readiness and ground infrastructure
- Data link infrastructure (Link 16, JREAP, IFF Mode 5)

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| SATCOM ground segment | Is MILSATCOM ground infrastructure compatible with MRTT-Q300? |
| Crypto infrastructure | Are crypto key management systems (EKMS) in place? |
| Mission data loading | Is ground-based MDS infrastructure available at all operating locations? |
| S1000D data hosting | Is secure technical data hosting aligned with MRTT-Q300 CSDB? |
| Quantum comms readiness | Is ground infrastructure quantum-communications-ready? |
