# D — Defense | I — Infrastructures | 08 — Dual-Use Civil-Defense Interface Infrastructure
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **dual-use civil-defense interface infrastructure** for the AMPEL360 MRTT Q300 programme. It covers the interoperability infrastructure between military operations and civil aviation systems, airport infrastructure shared with civil operations, humanitarian assistance/disaster relief (HA/DR) interface points, and civil authority coordination mechanisms.

## 2. Scope

- Civil airport compatibility and access agreements for MRTT-Q300 transit and recovery operations
- Civil air traffic control (ATC) integration and civil/military airspace coordination
- Humanitarian assistance and disaster relief (HA/DR) infrastructure interfaces
- Civil-military fuel and ground handling service agreements
- Passenger/cargo transport role interface with civil terminal infrastructure
- Dual-use certification and airworthiness authority coordination

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| Civil airport compatibility | Is MRTT-Q300 compatible with civil airport infrastructure at key locations? |
| ATC interoperability | Can MRTT-Q300 integrate seamlessly with civil ATC procedures? |
| HA/DR interface | Is infrastructure defined for humanitarian mission interface with civil agencies? |
| Civil fuel access | Can the aircraft use civil fuel supply chains for SAF-capable operations? |
| Dual-use certification | Is civil/military airworthiness coordination infrastructure in place? |
