# D — Defense | I — Infrastructures | 05 — Logistics, Mobility and Deployment Infrastructure
## AMPEL360 MRTT Q300 (MRTT-Q300)

## 1. Purpose

This folder assesses **logistics, mobility, and deployment infrastructure** for the AMPEL360 MRTT Q300 programme. It covers the strategic logistics chain, expeditionary deployment support, pre-positioning infrastructure, and theater-entry logistics enabling MRTT-Q300 operations.

## 2. Scope

- Strategic airlift and sealift integration for MRTT-Q300 deployment packages
- Forward logistics element (FLE) and deployed logistics support infrastructure
- Pre-positioning of fuel, spares, and consumables at forward locations
- Expeditionary airfield infrastructure compatibility (bare-base operations)
- Inter-agency and coalition logistics coordination infrastructure

## 3. Key Assessment Areas

| Assessment Area | Primary Question |
|----------------|-----------------|
| Strategic deployment | Can MRTT-Q300 self-deploy to all required theater locations? |
| Bare-base compatibility | Is the aircraft compatible with expeditionary airfield standards? |
| FLE infrastructure | Is forward logistics element infrastructure defined? |
| Pre-positioning | Are pre-positioning agreements and infrastructure in place? |
| Coalition logistics | Does logistics infrastructure support coalition operations? |
