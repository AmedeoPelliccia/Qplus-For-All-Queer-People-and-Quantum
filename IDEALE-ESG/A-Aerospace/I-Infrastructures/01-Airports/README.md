# 01 — Airports

> **01-Airports** defines the controlled infrastructure layer for airport-side aerospace operations, covering aircraft compatibility, ground handling, servicing, passenger interfaces, safety response, energy/refuelling systems, digital operations, and certification evidence.

## Purpose

Airport infrastructure within the `I-Infrastructures` axis of `A-Aerospace`.
Covers civil airport facilities, runways, terminals, ATC infrastructure, and ground support systems.
