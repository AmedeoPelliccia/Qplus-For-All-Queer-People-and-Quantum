# ATLAS 050-059 · 05.050 — General

**Folder path:** `Q+ATLANTIDE/000-099_ATLAS/050-059_Estructuras/050_General/`

## Purpose

This subsection provides the overarching general description of the structural architecture of the Q+ATLANTIDE aircraft family. It covers structural zoning, classification, load environment, applicability/effectivity rules, maintenance concept, inspection programme overview, structural health monitoring interfaces, and S1000D/CSDB mapping applicable to all structures subsections (050–059).
