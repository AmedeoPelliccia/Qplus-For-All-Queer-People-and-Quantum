# ATLAS 030-039 · Section 03 · Subsection 036 · 070 — Pneumatic Ground Service and Test Interfaces

## 1. Purpose

Documents the ground high-pressure air connection panel, bleed-air system ground test procedures, leak check provisions, and interface with ground support equipment (GSE).

## 2. Scope

- Ground HP coupling location, access door, and pressure limit placard.
- System-level leak test procedure: pressurisation source, hold time, and acceptance criteria.
- Valve-position verification procedure with aircraft power-on and ground test box.
- Interface with Aircraft Maintenance Manual (AMM) task references.
- Not in scope: engine ground run requirements (ATA 72) or APU start procedures (ATA 49).
