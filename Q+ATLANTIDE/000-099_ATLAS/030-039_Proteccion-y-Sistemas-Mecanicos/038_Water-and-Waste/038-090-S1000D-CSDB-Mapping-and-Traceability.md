# ATLAS 030-039 · Section 03 · Subsection 038 · 090 — S1000D CSDB Mapping and Traceability

## 1. Purpose

Controlled mapping of ATA 38 subsubjects (038-000 through 038-080) to S1000D Issue 6.0 Data Module Codes (DMC), CSDB publication objects, and applicable ATA iSpec 2200 cross-references.

## 2. Scope

- DMC model-identification code assignment for ATA 38 data modules.
- System/sub-system/unit breakdown table (SNS ↔ DMC system/sub-system codes).
- Publication module (PM) and applicability cross-reference list placeholders.
- Traceability matrix from ATLAS subsubject codes to iSpec 2200 subject/task codes.
- Not in scope: S1000D XML authoring rules or CSDB configuration management.
