# DTTA 200-209 · Section 00 · Subsection 202 — Armamento Convencional: Clasificación y Control

## §1 Purpose

This subsection establishes the Q+ATLANTIDE[^baseline] controlled documentation node for **Armamento Convencional: Clasificación y Control** within the Defence Technology Type Architecture (DTTA)[^archtable], code range 200-209, subsection 202.

**Non-operational boundary:** This subsection is restricted to classification, governance, custody, safety, accountability and legal-control taxonomy. It does not define construction details, deployment methods, targeting logic, tactical employment, optimization for harm, performance enhancement or operational weapon procedures.

**Safety boundary:** Conventional armament control is defence-critical and legally restricted; it requires explicit authorization, human control, safe custody, inventory accountability, hazard prevention, export-control review, non-proliferation compliance and lifecycle traceability.

All documents in this subsection are issued under governance class `restricted`[^gov] per rule N-006[^n006] and must not be distributed outside authorized Q+ATLANTIDE channels.

## §2 Scope

Subsection 202 covers the following subject areas:

- Controlled definition and disambiguation of conventional armament within DTTA taxonomy.
- Non-operational classification of armament classes (abstract taxonomy labels only).
- Platform compatibility interface taxonomy and integration boundary declarations.
- Storage, handling and safety governance frameworks.
- Inventory accountability and chain-of-custody governance taxonomy.
- Authorization framework, rules-of-use taxonomy and mandatory human control requirements.
- Risk classification and hazard prevention taxonomy.
- Export control, legal framework mapping and non-proliferation obligation taxonomy.
- Evidence, audit and compliance record frameworks.
- Traceability, evidence package and lifecycle governance.
