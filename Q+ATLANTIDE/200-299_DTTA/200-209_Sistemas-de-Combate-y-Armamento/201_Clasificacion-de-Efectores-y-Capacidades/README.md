# DTTA 200-209 · Section 00 · Subsection 201 — Clasificación de Efectores y Capacidades

## §1 Purpose

This subsection provides the taxonomy, governance, and assurance classification framework for effectors and defence capabilities within DTTA Section 00 (Sistemas de Combate y Armamento), code range 200-209.

**Non-operational boundary:** This subsection classifies effectors and defence capabilities at taxonomy, governance and assurance level only. It does not define performance optimization, targeting, employment tactics, deployment methods, construction parameters or operational procedures.

## §2 Scope

Subsection 201 covers:

- Controlled definitions of "effector" and "defence capability" within the Q+ATLANTIDE baseline.
- Non-operational taxonomy of effector classes and defensive capability categories.
- Boundary declarations between kinetic, non-kinetic, and support capabilities.
- Platform integration authority interfaces (governance taxonomy only).
- Human control requirements, rules-of-use taxonomy, and authorization gate structures.
- Safety classification, risk taxonomy, and collateral harm prevention boundaries.
- Interoperability mappings to NATO, EU, and international standards.
- Export control, legal, and ethical constraint framework.
- Traceability, evidence package obligations, and lifecycle governance.
