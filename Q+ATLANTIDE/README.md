## 3. Invariants

- **Code range:** `000-999` (no `1000` operational band; ATLAS-1000 is an umbrella register, not a numeric band).
- **No AAA domain:** true — use *Programme / Q-Division Interface* instead.
- **Architecture is not department:** true — bands classify *what* a technology is.
- **Q-Divisions are technical centres:** they own engineering authority (*who*).
- **ORB-Functions are enterprise support:** they provide enterprise services, not technical ownership.
- **Ranges must not overlap:** each band owns a disjoint slice of `000-999`.
